## 💰 定价说明

### 计费方式

- **按量计费**：根据实际使用的 Token 数量计费
- **无最低消费**：充多少用多少，余额永不过期
- **实时扣费**：每次调用后立即从余额扣除费用

### 价格优势

- 官方源头转发，价格略有优势
- 批量使用可联系客服获取更优惠价格
