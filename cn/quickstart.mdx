## 第一步：账户准备

### 1.1 注册账号

访问 [Nebula Api 官网](https://ai-nebula.com) 注册账号：
- 使用手机号注册并完成验证
- 登录控制台

### 1.2 账户充值

在控制台进行充值：
1. 点击「个人中心」
2. 点击「费用充值」，完成支付（支持支付宝、微信）

<Note>
新用户注册额外赠送 1 美金
</Note>

## 第二步：创建 API 密钥

### 2.1 生成密钥

1. 在后台导航点击「模型中心」
2. 点击「API 密钥」
3. 点击「新建」生成密钥

<Warning>
保存好您的 API 密钥，密钥只会显示一次
</Warning>

## 第三步：开始调用

### 3.1 获取接入信息

| 配置项 | 值 |
|-------|-----|
| **API 地址（Base URL）** | `https://llm.ai-nebula.com/v1` |
| **API 密钥** | 您创建的密钥 |
| **请求格式** | 完全兼容 OpenAI API |

### 3.2 测试调用

使用 curl 快速测试：

```bash
curl https://llm.ai-nebula.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

### 3.3 代码示例

<Tabs>
  <Tab title="Python">
    ```python
    from openai import OpenAI

    client = OpenAI(
        api_key="YOUR_API_KEY",
        base_url="https://llm.ai-nebula.com/v1"
    )

    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "user", "content": "Hello!"}
        ]
    )

    print(response.choices[0].message.content)
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    import OpenAI from 'openai';

    const openai = new OpenAI({
      apiKey: 'YOUR_API_KEY',
      baseURL: 'https://llm.ai-nebula.com/v1'
    });

    const response = await openai.chat.completions.create({
      model: 'gpt-4o',
      messages: [{ role: 'user', content: 'Hello!' }]
    });

    console.log(response.choices[0].message.content);
    ```
  </Tab>
  <Tab title="Java">
    ```java
    OpenAiService service = new OpenAiService(
        "YOUR_API_KEY", 
        Duration.ofSeconds(60), 
        "https://llm.ai-nebula.com/v1"
    );

    ChatCompletionRequest request = ChatCompletionRequest.builder()
        .model("gpt-4o")
        .messages(List.of(
            new ChatMessage(ChatMessageRole.USER, "Hello!")
        ))
        .build();

    ChatCompletionResult result = service.createChatCompletion(request);
    System.out.println(result.getChoices().get(0).getMessage().getContent());
    ```
  </Tab>
</Tabs>
