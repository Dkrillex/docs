## 一、简介

沉浸式翻译是一款优秀的浏览器翻译扩展，支持网页双语对照阅读。通过集成 **Nebula Api**，可调用强大的 AI 模型，获得更准确、自然的翻译效果。

## 二、快速安装

### 支持的浏览器

- Chrome / Edge / Brave
- Firefox
- Safari

### 安装步骤

1. 访问对应浏览器的扩展商店
2. 搜索"沉浸式翻译"或"Immersive Translate"
3. 点击"安装"并添加到浏览器工具栏
4. 或访问 [沉浸式翻译官网](https://immersivetranslate.com)

## 三、配置 Nebula Api

1. **打开设置**：点击浏览器工具栏的扩展图标，选择"设置"
2. **选择翻译服务**：在左侧菜单点击"翻译服务"，找到"OpenAI"
3. **填写配置**：
   - **APIKEY**：您的 Nebula Api 密钥
   - **自定义 API 接口地址**：`https://llm.ai-nebula.com/v1/chat/completions`
   - **自定义模型**：可选填（如 `gpt-3.5-turbo`）

## 四、核心功能

### 1. 网页翻译

**自动翻译**
- 访问外文网页时自动检测语言
- 点击工具栏"翻译"按钮完成整页翻译

**手动翻译**
- 点击扩展图标，选择"翻译本页"

### 2. 翻译模式

**双语对照（推荐）**
- 保留原文格式，译文显示在原文下方
- 适合对照学习、核对专业术语

**仅译文**
- 完全替换原文，适合快速阅读

### 3. 划词翻译

- 选中文本，点击弹出的"翻译"按钮
- 支持复制、发音等功能

## 六、特色功能

### 1. PDF 翻译

- 支持在线 PDF 文档翻译
- 保持原格式，双语对照显示

### 2. 视频字幕翻译

支持 YouTube、Netflix、Bilibili 等：
1. 开启"视频字幕翻译"功能
2. 选择字幕显示方式
3. 调整字幕样式

### 3. 电子书翻译

- 上传 EPUB 格式电子书
- 选择翻译语言和模式
- 下载双语版本

### 4. 输入框翻译

- 在输入框输入文本
- 按 `Alt + S` 触发翻译
- 一键替换为译文

## 七、快捷键

| 功能 | 默认快捷键 |
|------|-----------|
| 翻译/显示原文 | `Alt + T` |
| 切换翻译模式 | `Alt + M` |
| 翻译选中文本 | `Alt + S` |
| 打开设置 | `Alt + O` |

## 八、模型选择建议

| 内容类型 | 推荐模型 | 原因 |
|---------|---------|------|
| 新闻文章 | `gpt-3.5-turbo` | 快速、准确，性价比高 |
| 技术文档 | `gpt-4` | 术语翻译精准 |
| 学术论文 | `claude-3-opus` | 长文本理解力强 |
| 文学作品 | `claude-3-sonnet` | 保留文学美感 |
| 日常网页 | `gpt-3.5-turbo` | 响应快，满足基础需求 |

## 九、性能优化

### 1. 缓存设置

- 开启"翻译缓存"功能
- 设置缓存时长（建议 24 小时）

### 2. 批量翻译

- 调整批量大小（5-10 段落/批）
- 设置合理并发数（2-3 个）

### 3. 触发条件

- 最小翻译长度：10 个字符
- 忽略导航菜单、广告等元素
- 延迟翻译：200ms

## 十一、最佳实践

### 阅读体验优化

- 选择清晰字体（微软雅黑、思源黑体）
- 提高译文与原文颜色对比度
- 设置行间距 1.5 倍

### 学习辅助

- 使用"双语对照"模式对照学习
- 用划词翻译查询专业术语
- 导出翻译内容用于复习

### 成本控制

- 日常浏览用 `gpt-3.5-turbo`
- 重要文档用 `gpt-4` 或 Claude
- 利用缓存减少重复翻译
