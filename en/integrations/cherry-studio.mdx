## Introduction

Cherry Studio is a powerful AI chat client that supports multiple large language models. With **Nebula Lab**, you can easily use various mainstream AI models in Cherry Studio.

## Quick Integration

### 1. Get API Key

Visit [Nebula Lab](https://ai-nebula.com) to get your API key.

### 2. Configuration

1. Open Cherry Studio
2. Click the settings icon on the left
3. Select "Model Service"
4. Fill in:
   - API URL: `https://llm.ai-nebula.com`
   - API Key: Your key from Nebula Lab console
5. Click "âž• Add" to save

<Warning>
API URL must be `https://llm.ai-nebula.com`
</Warning>

### 3. Add Models

1. Search for target model (e.g. `gemini-2.5-pro-preview-06-05`)
2. Click the icon next to model name to configure
3. Enable/disable model variants as needed
4. Use right-side buttons for detailed configuration

## Advanced Features

### Image Support

For vision-enabled models (e.g. `GPT-4V`):

1. Enable "Images" in settings
2. Select a vision-capable model
3. Upload images directly in chat

### Streaming Output

Cherry Studio supports streaming output by default for smoother conversations.
