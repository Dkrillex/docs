## Step 1: Account Setup

### 1.1 Register

Visit [Nebula Api](https://ai-nebula.com) to register:
- Sign up with your mobile number and verify
- Log in to the console

### 1.2 Add Credits

Top up in the console:
1. Click "Personal Center"
2. Click "Billing Recharge" to pay (Alipay, WeChat supported)

<Note>
New users receive a $1 bonus upon registration
</Note>

## Step 2: Create API Key

### 2.1 Generate Key

1. Click "Model Center" in the navigation
2. Click "API Keys"
3. Click "Create New" to generate a key


## Step 3: Start Calling

### 3.1 Connection Info

| Config | Value |
|--------|-------|
| **API URL (Base URL)** | `https://llm.ai-nebula.com/v1` |
| **API Key** | Your created key |
| **Request Format** | Fully OpenAI API compatible |

### 3.2 Test Call

Quick test with curl:

```bash
curl https://llm.ai-nebula.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

### 3.3 Code Examples

<Tabs>
  <Tab title="Python">
    ```python
    from openai import OpenAI

    client = OpenAI(
        api_key="YOUR_API_KEY",
        base_url="https://llm.ai-nebula.com/v1"
    )

    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[
            {"role": "user", "content": "Hello!"}
        ]
    )

    print(response.choices[0].message.content)
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    import OpenAI from 'openai';

    const openai = new OpenAI({
      apiKey: 'YOUR_API_KEY',
      baseURL: 'https://llm.ai-nebula.com/v1'
    });

    const response = await openai.chat.completions.create({
      model: 'gpt-4o',
      messages: [{ role: 'user', content: 'Hello!' }]
    });

    console.log(response.choices[0].message.content);
    ```
  </Tab>
  <Tab title="Java">
    ```java
    OpenAiService service = new OpenAiService(
        "YOUR_API_KEY", 
        Duration.ofSeconds(60), 
        "https://llm.ai-nebula.com/v1"
    );

    ChatCompletionRequest request = ChatCompletionRequest.builder()
        .model("gpt-4o")
        .messages(List.of(
            new ChatMessage(ChatMessageRole.USER, "Hello!")
        ))
        .build();

    ChatCompletionResult result = service.createChatCompletion(request);
    System.out.println(result.getChoices().get(0).getMessage().getContent());
    ```
  </Tab>
</Tabs>

## FAQ

<AccordionGroup>
  <Accordion title="How to switch models?">
    Just change the `model` parameter in your request:
    ```json
    {
      "model": "gpt-4o",
      "model": "claude-sonnet-4-20250514",
      "model": "gemini-2.5-pro"
    }
    ```
  </Accordion>
  <Accordion title="Which programming languages are supported?">
    Nebula Api is OpenAI API compatible, supporting all OpenAI SDK languages:
    - Python
    - JavaScript/TypeScript
    - Java
    - C#/.NET
    - Go
    - Ruby
    - PHP
  </Accordion>
  <Accordion title="How to check balance?">
