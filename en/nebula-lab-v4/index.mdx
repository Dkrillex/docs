## Introduction

Nebula Lab V4 API provides various functional interfaces for video and image generation, including text-to-video, image-to-video, text-to-image, image-to-image, and other AI creative capabilities. All interfaces support unified authentication and consistent response formats, facilitating quick integration for developers.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer Token, e.g., `Bearer sk-xxxxxxxxxx`
</ParamField>

## Response Format

All interfaces return a unified JSON format:

<ResponseExample>
```json
{
  "code": 200,
  "msg": "Operation successful",
  "data": {
    // Interface specific return data
  }
}
```
</ResponseExample>

### Error Response

<ResponseExample>
```json
{
  "code": 400,
  "msg": "Parameter error",
  "data": null
}
```
</ResponseExample>
