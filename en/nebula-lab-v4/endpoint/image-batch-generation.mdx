## Introduction

Submit batch image generation task.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer Token, e.g., `Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...`
  How to get: Obtain via the POST interface in the Authentication module at the top left, `https://docs.openai-nebula.com/en/nebula-lab-v4/endpoint/get-token`
</ParamField>

## Request Parameters

<ParamField body="req.prompt" type="string" required>
  Image description prompt
</ParamField>

<ParamField body="req.model" type="string" required>
  Generation model ID
</ParamField>

<ParamField body="req.n" type="integer">
  Number of images to generate
</ParamField>

<ParamField body="req.size" type="string">
  Image size
</ParamField>
